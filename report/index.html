장시작 리포트

예정 업데이트 시각: 08:30

실제 업데이트 일시 YYYY-MM-DD HH:MM KST
상태 발행 전 / 발행 완료 / 수정 반영

1) 오늘 핵심 테마

  • 당일 시장의 가장 중요한 흐름을 한 줄로 정리
  • 장 시작 전 집중해야 할 방향성만 적기

2) 후보 종목

  • 종목명 / 핵심 근거 / 기대 방향
  • 근거 뉴스는 총 5개만, 제목 클릭 시 원문 이동

3) 제외 종목

  • 장 초반 제외해야 할 종목과 이유

4) 리스크

  • 지수, 수급, 이벤트, 변동성 위험

5) 오늘 운영 포인트

  • 시초 / 눌림 / 추세 전개 중 우선순위

뉴스는 각 항목에 원문 링크를 달아 5개만 유지

예정 업데이트 시각: 15:30

실제 업데이트 일시 YYYY-MM-DD HH:MM KST
상태 발행 전 / 발행 완료 / 수정 반영

1) 오늘 실제로 강했던 테마

  • 시장 반응이 좋았던 섹터와 이유

2) 실패한 테마 / 주도 종목

  • 기대 대비 약했던 흐름
  • 주도 종목과 실패 종목 비교

3) 시장 리스크 복기

  • 장중 변동성, 수급 왜곡, 뉴스 영향

4) 내일 체크포인트

  • 다음 거래일에 이어볼 핵심 관찰 포인트

5) 관련 뉴스 5개

  • 뉴스 제목 1 — 원문 링크
  • 뉴스 제목 2 — 원문 링크
  • 뉴스 제목 3 — 원문 링크
  • 뉴스 제목 4 — 원문 링크
  • 뉴스 제목 5 — 원문 링크

항상 유지되는 공개 리포트 규칙

  • 장시작 / 장종료 두 종류만 공개
  • 가장 최근 업데이트가 항상 상단 기준

  • 근거 뉴스는 각 리포트당 5개만 유지
